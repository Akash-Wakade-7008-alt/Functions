# 📊 Mean Function (Java)

A simple Java-based implementation to calculate the **average (mean)** of two integer numbers. This project is part of my journey of translating mathematical concepts into code.

---

## 🚀 Overview

This program takes **two integer inputs from the user** and computes their **mean (average)** using a custom function.

It’s designed to be:

* Beginner-friendly
* Easy to understand
* Focused on clarity of logic

---

## 🧠 Concept

The **mean (average)** of two numbers is calculated using the formula:

Mean = (a + b) / 2

Where:

* `a` = first number
* `b` = second number

---

## 💻 Features

* Accepts user input via console
* Uses a separate function for calculation
* Demonstrates basic Java concepts:

  * Classes & Objects
  * Methods
  * User Input (`Scanner`)

---

## 📂 Files

* `Main.java` → Handles user input and output
* `Mean.java` → Contains the logic for calculating the mean

---

## ▶️ How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/mean-function.git
   ```

2. Navigate to the project folder:

   ```bash
   cd mean-function
   ```

3. Compile the Java files:

   ```bash
   javac Main.java Mean.java
   ```

4. Run the program:

   ```bash
   java Main
   ```
